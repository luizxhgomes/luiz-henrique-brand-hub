01 · INTERNO
Núcleo de Inteligência de Dados V4 Bilinski
Sistema interno de orquestração operacional

Sistema que conecta 7 áreas da unidade V4 Bilinski (marketing, comunicação, comercial, CS/CX, operação, financeiro, RH) em fluxo único de dados. Agents de IA especializados operam sobre camada consolidada, orquestrador central coordena decisões. Operacionaliza o padrão Núcleo de Inteligência. Papel: arquiteto da evolução e gestor da criação, não idealizador único.

Stack: agents de IA orquestrados · camada unificada de dados · integrações multi-área · orquestrador central
EM PRODUÇÃO HÁ 8 MESES Interno V4 Bilinski & Co

6 Sistemas Autorais Externos

Construídos em paralelo, sem time de engenharia tradicional. Cada um prova a velocidade da Fábrica de Sistemas.

02
OUTBILI
Inteligência comercial e prospecção B2B

Prospecção B2B serverless com enriquecimento automático em cascata (CNPJa para dados cadastrais e sócios, Assertiva para WhatsApp confirmado do decisor). Gates obrigatórios de decisor mais telefone, pipeline Kanban com 7 estágios, campanhas WhatsApp integradas via BilinskiZap.

Stack: React 19 + TypeScript + Vite · Cloudflare Workers (proxy OAuth2) · Airtable · CNPJa API · Assertiva API · BilinskiZap API
EM PRODUÇÃO v4bilinski.github.io/outbili
03
BilinskiZap
Automação WhatsApp com IA integrada

Plataforma completa de automação WhatsApp. Campanhas em massa com templates Meta aprovados, inbox em tempo real no navegador, agents de IA que lembram do histórico de cada cliente, gestão de contatos via CSV com campos personalizados, lead forms embeddáveis.

Stack: API Oficial Meta · PWA · Agents com memória persistente · Templates sincronizados Meta
EM PRODUÇÃO Produto V4 Bilinski
04
Keyra
Financeiro operacional para estética

Primeiro financeiro operacional do setor de estética. Conecta agenda, serviços, comanda, transação e DRE automaticamente, eliminando alimentação manual pós-faturamento típica de ERPs. UX inegociável: números absolutos, comparativos textuais, tela única. Proposição: key para acelerar receita.

Stack: Next.js 16 (App Router + Server Actions) · Supabase (PostgreSQL com RLS multi-tenant) · Vercel · Inngest · Stripe · Asaas
EM PRODUÇÃO usekeyra.vercel.app
05
Social Media AI
Geração de conteúdo baseada em virais

Coleta Reels recentes de concorrentes, filtra os top virais por visualizações, analisa cada vídeo com Gemini (Conceito + Gancho + Retenção + Recompensa + Roteiro), envia análise mais contexto da marca para Claude gerar novos conceitos adaptados. Pipeline de 7 fases coordenadas.

Stack: Next.js 16 + TypeScript + Tailwind + shadcn/ui · Apify (coleta Instagram) · Google Gemini 2.0 Flash (análise multimodal) · Claude Sonnet (geração)
EM PRODUÇÃO Autoral
06
Carousel AI
Gerador de carrosséis Instagram com IA

Cria carrosséis multi-slide com texto e imagens geradas por IA. Templates estilizados (Twitter, Storyteller), customização de tema, drag-and-drop de slides, exportação direta para PNG. Integração Apify para referências visuais.

Stack: React 19 + TypeScript · Vite 6 · Tailwind 3 · shadcn/ui · Google Gemini (texto e imagem) · html-to-image · dnd-kit · Apify REST API
EM PRODUÇÃO Autoral
07
InsightMeet AI
Análise de linguagem não-verbal em reuniões

Analisa micro-expressões e gesticulações em reuniões de videoconferência para fornecer insights sobre dinâmica em tempo real. Identifica emoções (neutral, curiosidade, foco, skepticism, agreement, confusion, surprise), gestos, engajamento, atenção, postura, voz e dicção. Gera relatório para ata automaticamente. Vai além de transcrição: mapeia comunicação velada.

Stack: GCP Cloud Run · análise comportamental em tempo real · integração Google Meet · pipeline multimodal
EM PRODUÇÃO insightmeet-ai-721960084105.us-central1.run.app
