Prompts por Formato

FORMATO 1: POST INSTAGRAM
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um POST DE INSTAGRAM sobre [TEMA]. PARAMETROS: - Pilar: [AIOX / Bastidores / Storytelling / Educacao] - Objetivo: [Educar / Inspirar / Engajar / Converter] - Tom: [Reflexivo / Provocativo / Motivacional / Tecnico] - Fase atual (90 dias): [Declaracao / Documentacao / Consolidacao] TEMA ESPECIFICO: [Ex: "Hoje tentei fazer um agente analisar metricas e falhou 3 vezes"] REGRAS: - Maximo 2.200 caracteres - Primeira linha = gancho (pare o scroll) - Quebras de linha frequentes (legibilidade mobile) - Incluir pelo menos 1 frase-ancora do vocabulario proprietario - CTA no final - 5-10 hashtags relevantes - Tom: tecnico_humanizado_transparente
FORMATO 2: REEL / VIDEO CURTO
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um ROTEIRO DE REEL sobre [TEMA]. PARAMETROS: - Duracao: [30s / 60s / 90s] - Estilo: [Falando pra camera / Tela gravada / Mix] - Pilar: [AIOX / Bastidores / Storytelling / Educacao] ESTRUTURA OBRIGATORIA: [0-3s] GANCHO — frase que para o scroll [3-15s] CONTEXTO — situar quem voce e [15-Xs] CONTEUDO — entregar o valor [Ultimos 10s] CTA — o que o espectador deve fazer REGRAS: - Falar em primeira pessoa - Legendas curtas e impactantes - Indicar transicoes visuais - Incluir marcacao de energia (1-10)
FORMATO 3: CARROSSEL
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um CARROSSEL sobre [TEMA]. PARAMETROS: - Numero de slides: [5-10] - Pilar: [AIOX / Bastidores / Storytelling / Educacao] - Serie criativa: [Decodificando / Radiografia / Mito vs Real / Fluxograma / Nenhuma] ESTRUTURA OBRIGATORIA: SLIDE 1 (CAPA): Titulo provocativo + visual forte SLIDES 2-N: Um conceito por slide, texto curto SLIDE FINAL (CTA): Salva + comenta + segue REGRAS: - Maximo 40 palavras por slide - Cada slide deve funcionar sozinho (shareable) - Usar numeros e dados sempre que possivel - Visual: fundo escuro (#0A0E1A), texto claro, acentos em ciano/azul
FORMATO 4: POST LINKEDIN
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um POST LINKEDIN sobre [TEMA]. PARAMETROS: - Tipo: [Artigo / Opiniao / Case / Educativo / Storytelling] - Tom: [Mais formal que Instagram, mas ainda acessivel] - Audiencia: [Gestores / Profissionais de marketing / C-Level] ESTRUTURA OBRIGATORIA: LINHA 1: Gancho forte (aparece no preview) PARAGRAFO 1: Contexto e credibilidade CORPO: 3-5 paragrafos com insight + dado + exemplo FECHAMENTO: Pergunta aberta para engajamento HASHTAGS: 3-5 relevantes REGRAS: - Tom mais profissional que Instagram - Incluir dados e resultados mensuraveis - Parágrafos curtos (2-3 linhas max) - Usar quebras de linha para ritmo - Nao usar emojis em excesso (max 2-3)
FORMATO 5: THREAD X (TWITTER)
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere uma THREAD no X sobre [TEMA]. PARAMETROS: - Numero de tweets: [7-15] - Tipo: [Educativo / Opiniao / Storytelling] ESTRUTURA: TWEET 1: Gancho + promessa ("Thread sobre [X]:") TWEETS 2-N: Um ponto por tweet, numerados TWEET FINAL: Recap + CTA (RT + follow) REGRAS: - Max 280 caracteres por tweet - Cada tweet deve fazer sentido isolado - Numerar: "1/" "2/" etc - Usar dados quando possivel
FORMATO 6: STORIES INSTAGRAM
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere uma SEQUENCIA DE STORIES sobre [TEMA]. PARAMETROS: - Numero de stories: [4-6] - Story Set de referencia: [1-10 do Arsenal] - Objetivo: [Posicionar / Educar / Humanizar / Engajar] PARA CADA STORY GERAR: - TEXTO NA TELA (max 3 linhas) - VISUAL sugerido (tipo de midia) - NIVEL DE ENERGIA (1-10) - STICKER sugerido (enquete, pergunta, emoji slider) REGRAS: - Energia crescente (nunca decrescente) - Ultimo story = CTA claro - Alternar entre texto, selfie e prints - Incluir pelo menos 1 sticker interativo
