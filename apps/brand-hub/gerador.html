FORMATO 1: POST INSTAGRAM
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um POST DE INSTAGRAM sobre [TEMA]. PARÂMETROS: - Pilar: [Tese / Ensino / Portfólio / Narrativa] - Objetivo: [Educar / Provocar / Engajar / Converter] - Tom: [Arquiteto / Estrategista / Mentor / Construtor] - Fase (90 dias): [Clareza / Consistência / Autoridade] TEMA ESPECÍFICO: [Ex: "Hoje decidi que o squad de prospecção do OUTBILI vai ganhar um agent de qualificação extra"] REGRAS: - Máximo 2.200 caracteres - Primeira linha = gancho provocativo (inversão conceitual) - Quebras de linha frequentes (legibilidade mobile) - Incluir pelo menos 1 frase-âncora v6 (humanos orquestram, agents executam, etc) - Citar sistema específico do portfólio quando servir de prova - CTA no final (convite genuíno, não venda) - 5 a 10 hashtags proprietárias (#FabricaDeSistemas, #SquadFirst, #VibeCoding) - Zero travessões. Zero superlativos vazios.
FORMATO 2: REEL / VÍDEO CURTO
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um ROTEIRO DE REEL sobre [TEMA]. PARÂMETROS: - Duração: [30s / 60s / 90s] - Estilo: [Falando pra câmera / Tela gravada mostrando sistema / Split-screen antes e depois] - Pilar: [Tese / Ensino / Portfólio / Narrativa] - Sistema em destaque (se for Pilar Portfólio): [OUTBILI / BilinskiZap / Keyra / Social Media AI / Carousel AI / InsightMeet AI / Núcleo V4] ESTRUTURA OBRIGATÓRIA (building in public): [0-3s] GANCHO, inversão conceitual ("Sua empresa não tem problema de pessoas.") Comenta [palavra-chave] para o playbook." REGRAS: - Falar em primeira pessoa, posição de construtor - Mostrar evidência visual sempre que possível (print, vídeo de tela, dashboard) - Legendas curtas, tipografia limpa - Frase-âncora v6 na abertura ou encerramento - Zero pose de aprendiz ou "ainda estou descobrindo"

Checklist de Validação (8/9 = PUBLICA)

Todo conteúdo deve passar por este filtro antes de publicar.

01
Tom é provocativo e fundamentado (não guru, não hype)?
02
Tem prova concreta (sistema em produção, stack, dado verificável)?
03
Mostra posição de construtor (não de aprendiz)?
04
Tem gancho provocativo ou inversão conceitual?
05
É autêntico (não finge conhecimento que não tem)?
06
Inspira outros a construírem (alinha com missão de formação)?
07
Tem CTA claro?
08
Tom é decisivo mas acessível?
09
Usa pelo menos uma frase-âncora v6?
