Prompts por Formato

FORMATO 1: POST INSTAGRAM
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um POST DE INSTAGRAM sobre [TEMA]. PARÂMETROS: - Pilar: [AIOX / Bastidores / Storytelling / Educação] - Objetivo: [Educar / Inspirar / Engajar / Converter] - Tom: [Reflexivo / Provocativo / Motivacional / Técnico] - Fase atual (90 dias): [Declaração / Documentação / Consolidação] TEMA ESPECÍFICO: [Ex: "Hoje tentei fazer um agente analisar métricas e falhou 3 vezes"] REGRAS: - Máximo 2.200 caracteres - Primeira linha = gancho (pare o scroll) - Quebras de linha frequentes (legibilidade mobile) - Incluir pelo menos 1 frase-âncora do vocabulário proprietário - CTA no final - 5-10 hashtags relevantes - Tom: técnico_humanizado_transparente
FORMATO 2: REEL / VÍDEO CURTO
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um ROTEIRO DE REEL sobre [TEMA]. PARÂMETROS: - Duração: [30s / 60s / 90s] - Estilo: [Falando pra câmera / Tela gravada / Mix] - Pilar: [AIOX / Bastidores / Storytelling / Educação] ESTRUTURA OBRIGATÓRIA: [0-3s] GANCHO — frase que para o scroll [3-15s] CONTEXTO — situar quem você é [15-Xs] CONTEÚDO — entregar o valor [Últimos 10s] CTA — o que o espectador deve fazer REGRAS: - Falar em primeira pessoa - Legendas curtas e impactantes - Indicar transições visuais - Incluir marcação de energia (1-10)
FORMATO 3: CARROSSEL
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um CARROSSEL sobre [TEMA]. PARÂMETROS: - Número de slides: [5-10] - Pilar: [AIOX / Bastidores / Storytelling / Educação] - Série criativa: [Decodificando / Radiografia / Mito vs Real / Fluxograma / Nenhuma] ESTRUTURA OBRIGATÓRIA: SLIDE 1 (CAPA): Título provocativo + visual forte SLIDES 2-N: Um conceito por slide, texto curto SLIDE FINAL (CTA): Salva + comenta + segue REGRAS: - Máximo 40 palavras por slide - Cada slide deve funcionar sozinho (shareable) - Usar números e dados sempre que possível - Visual: fundo escuro (#0A0E1A), texto claro, acentos em ciano/azul
FORMATO 4: POST LINKEDIN
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere um POST LINKEDIN sobre [TEMA]. PARÂMETROS: - Tipo: [Artigo / Opinião / Case / Educativo / Storytelling] - Tom: [Mais formal que Instagram, mas ainda acessível] - Audiência: [Gestores / Profissionais de marketing / C-Level] ESTRUTURA OBRIGATÓRIA: LINHA 1: Gancho forte (aparece no preview) PARÁGRAFO 1: Contexto e credibilidade CORPO: 3-5 parágrafos com insight + dado + exemplo FECHAMENTO: Pergunta aberta para engajamento HASHTAGS: 3-5 relevantes REGRAS: - Tom mais profissional que Instagram - Incluir dados e resultados mensuráveis - Parágrafos curtos (2-3 linhas max) - Usar quebras de linha para ritmo - Não usar emojis em excesso (max 2-3)
FORMATO 5: THREAD X (TWITTER)
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere uma THREAD no X sobre [TEMA]. PARÂMETROS: - Número de tweets: [7-15] - Tipo: [Educativo / Opinião / Storytelling] ESTRUTURA: TWEET 1: Gancho + promessa ("Thread sobre [X]:") TWEETS 2-N: Um ponto por tweet, numerados TWEET FINAL: Recap + CTA (RT + follow) REGRAS: - Max 280 caracteres por tweet - Cada tweet deve fazer sentido isolado - Numerar: "1/" "2/" etc - Usar dados quando possível
FORMATO 6: STORIES INSTAGRAM
[Cole o PROMPT UNIVERSAL DE CONTEXTO acima] Agora gere uma SEQUÊNCIA DE STORIES sobre [TEMA]. PARÂMETROS: - Número de stories: [4-6] - Story Set de referência: [1-10 do Arsenal] - Objetivo: [Posicionar / Educar / Humanizar / Engajar] PARA CADA STORY GERAR: - TEXTO NA TELA (max 3 linhas) - VISUAL sugerido (tipo de mídia) - NÍVEL DE ENERGIA (1-10) - STICKER sugerido (enquete, pergunta, emoji slider) REGRAS: - Energia crescente (nunca decrescente) - Último story = CTA claro - Alternar entre texto, selfie e prints - Incluir pelo menos 1 sticker interativo

06

07

08

09
