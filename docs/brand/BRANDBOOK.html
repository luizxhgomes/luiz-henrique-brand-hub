REGRAS DE USO

O que fazer e o que nunca fazer com a identidade visual

FAZER

  • Usar dark mode como padrao (Midnight #0A0E1A)
  • Manter contraste minimo 4.5:1 em todo texto
  • Usar Montserrat 900 (Black) em MAIUSCULAS para titulos display
  • Aplicar gradiente accent em titulos hero e CTAs
  • Usar Montserrat 500 com letter-spacing para dados e metricas
  • Manter padding minimo de 48px nas laterais dos stories
  • Usar fotos reais (selfies, tela, setup) nos bastidores
  • Manter identidade visual consistente entre series
  • Usar iconografia Lucide (stroke 1.5px, cor cyan)
  • Respeitar safe zones dos stories (top 80px, bottom 200px)

NAO FAZER

  • Usar fundo branco como padrao (light mode so em LinkedIn mock)
  • Misturar mais de 3 cores de marca no mesmo slide
  • Usar peso 900 para texto corrido (reservar para titulos display)
  • Colocar emojis como icones principais
  • Usar fotos de banco de imagem genericas
  • Mudar paleta de cores entre posts da mesma serie
  • Usar sombras coloridas excessivas (glow sutil apenas)
  • Textos com mais de 40 palavras por slide de carrossel
  • Animacoes bouncing, zoom excessivo ou rotacao
  • Fontes decorativas ou script em qualquer contexto
