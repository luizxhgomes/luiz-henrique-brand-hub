## FRAMEWORKS PRÓPRIOS

### 1. Método SNI (Sistema Nervoso Inteligente)
Integração das 7 áreas operacionais com IA num único fluxo de dados. Materializado no Brabíssimo (17 módulos).

### 2. Framework GRA (Gargalo > Restrição > Agente)
Teoria das Restrições aplicada com agentes de IA:
- **Identificar:** Mapear fluxo completo de receita (lead > MQL > SQL > cliente > expansão > churn)
- **Explorar:** Extrair máximo do gargalo sem investir mais
- **Subordinar:** Alinhar demais áreas ao ritmo do gargalo
- **Elevar:** Agente de IA resolve o gargalo autonomamente
- **Repetir:** Novo gargalo emerge, ciclo recomeça

### 3. Equação OI (Operações Inteligentes)
```
RESULTADO = (PESSOAS x PROCESSOS x DADOS) ^ IA
```

### 4. Escada de Maturidade IA
- **Nível 1:** Uso básico (ChatGPT, copilots)
- **Nível 2:** Automação (n8n, Make, Zapier)
- **Nível 3:** Agentes autônomos (execução sem supervisão)
- **Nível 4:** Núcleo inteligente (7 áreas integradas, IA operacional)
